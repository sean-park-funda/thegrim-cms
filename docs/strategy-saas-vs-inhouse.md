# 더그림 CMS: 인하우스 강화 vs B2B SaaS 전략 검토
> 카니발라이제이션 이슈 + 시장 규모 분석 | 2026년 4월

---

## 결론 먼저

**단기 답변**: B2B SaaS 시장은 예상보다 훨씬 좁다 ($25M 글로벌). 본격적인 SaaS 사업은 전혀 다른 조직 역량을 요구한다. **카니발 위험은 조건부로 통제 가능하지만, 시장이 좁아서 수지타산이 맞지 않을 수 있다.**

**전략적 권고**: 완전한 SaaS 피벗보다 **"인하우스 완성 → 선별적 라이선싱"** 순서가 현실적이다.

---

## 1. 카니발라이제이션 — 실제로 발생하는가?

### 카니발은 분명히 존재한다

앞서 Epic/ILM/WetaFX 사례를 들어 "카니발 없음"이라고 썼으나 이는 **더그림에 적용하면 틀린 비유**다.

- Epic: Fortnite 플레이어 ≠ Unreal 라이선시 → 고객군이 다름
- ILM: 영화 관객 ≠ StageCraft 사용 VFX 스튜디오 → 고객군이 다름
- **더그림**: 웹툰 제작사 = CMS 잠재 고객 → **고객군이 동일**

레드아이스, 재담미디어 같은 B2B 구매자는 더그림과 **같은 플랫폼(네이버웹툰 등)에서 독자를 두고 직접 경쟁하는 제작사들**이다. 이들이 더그림 CMS로 제작 효율이 오르면 더 많은 작품을 더 빠르게 출시하고, 플랫폼 노출과 독자 점유율에서 더그림과 경쟁하게 된다. **카니발은 실제로 존재한다.**

### 그러면 카니발의 실질적 크기는 어느 정도인가

카니발이 있다는 사실보다 중요한 건 **얼마나 치명적이냐**다. 이는 더그림의 진짜 경쟁력이 무엇이냐에 달려있다.

| 시나리오 | 도구 판매 후 경쟁사 변화 | 더그림 피해 수준 |
|----------|------------------------|----------------|
| **A. 더그림 해자 = 박태준 IP** | 경쟁사가 빨라지지만 박태준을 복제 못함 | 낮음 — 감내 가능 |
| **B. 더그림 해자 = 제작 효율** | 경쟁사가 동등한 효율 획득 | 높음 — 치명적 |
| **C. 도구에 박태준 스타일 포함** | 경쟁사가 유사 스타일 IP 양산 | 매우 높음 — 불가 |

**현실적 판단**: 더그림의 해자가 박태준 브랜드·파주 세계관·팬덤에서 나온다면, 경쟁사가 제작 도구를 가져도 그걸 복제하지는 못한다. 이 경우 카니발이 있어도 치명적이지 않다.

### 단, 한 가지 중요한 반론

경쟁사들은 **어차피 AI 도구를 갖게 된다.** WeToon, TooToon, Jenova AI 등 외부 도구가 이미 존재한다. 더그림이 CMS를 팔지 않아도 레드아이스는 다른 AI 도구로 효율화된다.

그렇다면 더그림이 파는 것과 안 파는 것의 실질적 차이는:
> **"더그림 특유의 노하우·워크플로우를 경쟁사에 넘기느냐"**

범용 AI 파이프라인 = 팔아도 카니발 효과 제한적 (어차피 다른 곳에서도 구할 수 있음)
더그림 고유 워크플로우·스타일 모델 = 절대 포함 불가

### 결론

카니발은 존재하지만 조건부로 감내 가능하다. 핵심 조건은 두 가지:
1. **박태준 스타일 모델·파주 세계관 모듈은 절대 포함하지 않는다**
2. **더그림의 경쟁력이 IP에서 온다는 확신이 있을 때만 진행**

그러나 카니발 감수 여부보다 더 중요한 문제가 있다 — **시장 자체가 너무 좁다.**

## 2. 시장 규모 — 예상보다 훨씬 좁다

### AI 웹툰·만화 제작 도구 시장

| 지표 | 수치 | 출처 |
|------|------|------|
| 글로벌 AI 만화 생성 시장 (2024) | **$25M** | Grand View Research |
| 글로벌 AI 만화 생성 시장 (2030) | **$102M** | Grand View Research |
| CAGR | 27.5% | - |
| Asia Pacific 점유율 | 42% | - |

> $25M 전체 시장에서 아시아 42% = $10.5M. 한국은 그 일부.
> 더그림이 한국 시장 점유율 30%를 가져가도 연 매출 **수십억원 미만**.

### 레퍼런스: Clip Studio Paint (Celsys)

가장 성공한 만화 제작 도구 전문 회사의 실제 수치.

| 지표 | 수치 |
|------|------|
| 구독 계약 수 (2023.12) | 94.5만 계약 |
| ARR (2023.12) | $21M (약 270억원) |
| 글로벌 멤버십 | 824만 명 |
| ARPU | 월 약 $1.9~$9.99 (티어별) |
| YoY 성장률 | 27.2% |

Clip Studio Paint는 전 세계 824만 명을 확보하고도 ARR이 $21M이다. **더그림이 도구 SaaS로 이 수준의 ARR을 달성하려면 수십만 구독자가 필요하다.**

### 한국 웹툰 산업 데이터

| 지표 | 수치 | 출처 |
|------|------|------|
| 한국 웹툰 산업 전체 매출 (2024) | 2.286조원 | KOCCA |
| 주요 플랫폼 등록 창작자 | 10만 명+ | 업계 추산 |
| AI 도구 활용 의향 사업체 | **63.8%** | KOCCA 2024 |
| 신규 데뷔 창작자/년 | 5,000명+ | 추산 |

**63.8% 활용 의향은 긍정적**이나, 의향과 실제 유료 구독은 다르다. 실제 전환율 가정(10%) 적용 시 유료 고객 수천 명 수준.

## 3. SaaS 사업의 숨겨진 비용 — 조직 역량 문제

WetaFX가 도구를 Unity에 매각한 진짜 이유를 보면:

> *"창작 비즈니스와 도구 SaaS는 전혀 다른 역량을 요구한다"*
> - 엔지니어링 조직 (버그 대응, 업데이트, API 관리)
> - 고객 지원 조직 (CS, 온보딩, 교육)
> - B2B 영업 조직 (계약, 가격 협상, 파트너십)
> - 마케팅 (개발자 관계, 컨퍼런스, 콘텐츠)

**더그림의 현재 핵심 역량**: 박태준 세계관 기반 콘텐츠 제작, 팬덤 관리, IP 확장
**SaaS에 필요한 역량**: 위 4가지 → 현재 더그림에 없음

이 역량을 구축하는 비용이 SaaS 매출을 장기간 초과할 가능성이 높다.

## 4. 전략 옵션 비교

### Option A: 완전 인하우스 (현재 방향)

- **장점**: 핵심 역량에 집중, 제작비 절감이 직접 수익으로
- **단점**: 개발 비용 회수 채널이 자체 작품 뿐
- **적합한 조건**: 더그림의 연간 작품 수가 충분히 많아서 ROI 달성될 때

**수익 시뮬레이션**:
- 현재 제작 인력 비용 절감 30% 가정 → 연 수억원 절감
- 작품 수 증가 효과 → IP 매출 증가 간접 기여

### Option B: 본격 B2B SaaS

- **장점**: 새로운 수익원, 시장 선점 가능성
- **단점**: 좁은 시장($25M 글로벌), 새로운 조직 역량 필요, 카니발 위험 통제 필요
- **적합한 조건**: 전담 팀 구성 가능하고 2~3년 적자 감수 가능할 때

**수익 시뮬레이션**:
- 한국 스튜디오 50개 × 월 100만원 = 연 6억원
- 글로벌 확장 시 5배 = 연 30억원 (3~5년 후)
- 더그림 현재 매출(248억원) 대비 미미한 수준

### Option C: 선별적 라이선싱 (권고안)

도구를 직접 SaaS로 팔지 않고, **전략적 파트너에게 선별적으로 라이선싱**. SaaS 조직 없이도 수익화 가능.

```
더그림 CMS 핵심 기능
├── 인하우스 전용 (공개 불가)
│   ├── 박태준 스타일 LoRA 모델
│   ├── 파주 세계관 캐릭터 일관성 모듈
│   └── 영업 기밀 워크플로우
│
├── 라이선싱 가능 (선별 파트너)
│   ├── 웹툰→영상 변환 파이프라인
│   ├── AI 채색 모듈
│   └── 컷 배치 자동화
│
└── 파트너십 제안 대상
    ├── 레드아이스 스튜디오 (비경쟁: 웹소설 원작)
    ├── 메타툰 (비경쟁: 글로벌 유통 특화)
    └── 일본 파트너 (지역 비경쟁)
```

- **장점**: SaaS 조직 불필요, 카니발 위험 통제, 수익화 가능
- **단점**: 규모화 어려움, 파트너 관리 필요

## 5. 진짜 기회는 어디에 있나

시장 데이터를 보면 도구 SaaS보다 **IP 활용 영상화**가 훨씬 큰 시장이다.

| 시장 | 규모 (2025) | 성장률 |
|------|------------|--------|
| AI 웹툰 제작 도구 시장 | $25M | 27.5% |
| 글로벌 웹툰 콘텐츠 시장 | $8.5B | 24.3% |
| 애니메이션 AI 생성 시장 | $2.37B | 36% |
| **비율** | **1 : 340 : 95** | - |

도구 시장은 콘텐츠 시장의 **1/340** 수준. 더그림이 만들어야 할 것은 도구가 아니라, 더그림 CMS를 내부에서 활용해서 만든 **콘텐츠**다.

**가장 큰 기회**: AI 영상화 파이프라인을 내재화해서 외모지상주의·싸움독학의 **단편 애니메이션·숏폼 영상을 빠르고 저렴하게 대량 제작** → 유튜브/인스타릴스/틱톡 배포 → IP 글로벌 인지도 상승 → 2차 IP 수익(애니화 계약, 굿즈, 게임)

## 6. 요약 및 제언

### 카니발 이슈 결론
카니발은 실제로 존재한다. B2B 고객(경쟁 제작사)이 더 효율적이 되면 더그림과 직접 경쟁한다. 단, 더그림의 해자가 박태준 IP라면 도구 공유만으로 복제는 불가능하고, 어차피 경쟁사들은 외부 AI 도구로 효율화되므로 카니발의 **증분 효과는 제한적**이다. 박태준 스타일 모델과 파주 세계관 모듈은 절대 포함하지 않는다는 조건이 전제돼야 한다.

### 시장 규모 결론
**AI 웹툰 도구 시장은 $25M으로 협소하다.** Clip Studio Paint 같은 전문 회사도 전 세계 824만 명으로 ARR $21M이다. 이 시장에서 더그림이 의미있는 SaaS 매출을 올리기 어렵다.

### 전략 우선순위 (권고)

```
1순위: 인하우스 완성
  → AI 파이프라인으로 더그림 작품 제작 원가 절감
  → 작품 수 증가, 숏폼 대량 배포로 글로벌 IP 확대

2순위: 선별적 라이선싱
  → SaaS 조직 없이, 비경쟁 스튜디오에 파이프라인 일부 라이선싱
  → 연 수억~십수억원 부수입 가능

3순위: SaaS 피벗 (장기, 조건부)
  → 인하우스 ROI 검증 후 + 전담팀 구성 가능할 때
  → 시장이 $100M+ 규모로 성장한 2028~2029년 재검토
```

### 핵심 판단 기준
> **더그림의 브랜드 가치는 박태준 작가와 파주 세계관에서 온다. 도구는 수단이다.**
> 도구를 잘 만드는 것이 목적이 아니라, 도구를 통해 더 좋은 IP를 더 빠르게 만드는 것이 목적이다.
> WetaFX는 도구 사업 자체를 매각하고 창작에 집중했다.
