## 1. 양사 현황 요약

### 하이버 (Hiver) — 남성 패션 쇼핑 플랫폼

| 항목 | 내용 |
|------|------|
| 슬로건 | "1000만 남자들의 패션앱" |
| 서비스 출시 | 2018년 11월 |
| 운영사 | (주)하이버 / 모회사: 뉴넥스(Newnex) |
| 회원 수 | 375만 명 |
| 핵심 타깃 | 20~30대 MZ세대 남성 |
| 매출 | 571억 원 (2023년) → 195억 원 (2024년, 급감) |
| 현재 상태 | **기업회생 절차 중** (2025년 9월 뉴넥스 신청, 서비스는 정상 운영) |

**강점:**
- 남성 버티컬 커머스 1위 포지션
- 365일 무료배송 + 하루배송 (남성 커머스 최초)
- 100만 개+ 상품, 스트리트·럭셔리·스포츠·그루밍 전 카테고리
- 3년+ 남성 쇼핑 데이터 → 맞춤 큐레이션 알고리즘

**현재 과제:**
- 기업회생으로 인한 브랜드 신뢰도 하락
- 마케팅 예산 제약 (임직원 3명 수준으로 슬림화)
- 화제성 회복 및 리포지셔닝 필요

## 2. 핵심 시너지 — 타깃이 완벽하게 일치한다

**하이버 고객 = 더그림 독자**

| 구분 | 하이버 | 더그림 |
|------|--------|--------|
| 타깃 성별 | 남성 95%+ | 외모지상주의·싸움독학 독자 남성 다수 |
| 타깃 연령 | 20~30대 MZ | 웹툰 핵심 독자층 20~30대 |
| 관심사 | 패션·스타일·스트리트컬처 | 외모·패션·액션·성장 서사 |
| 플랫폼 | 모바일 앱 | 모바일 웹툰 |

더그림의 대표작 *외모지상주의*는 **외모·패션·스타일이 핵심 주제**인 작품이다.  
하이버가 파는 것과 이 웹툰이 다루는 것이 본질적으로 같다.

## 3. 콜라보레이션 시나리오

### Tier 1: 즉시 추진 (1~3개월, 저비용 고효과)

#### 👗 A. 외모지상주의 × 하이버 캐릭터 코디 콘텐츠

- **구조**: 외모지상주의 주요 캐릭터(박형석, 지호 등)의 실제 코디를 하이버 입점 상품으로 재현
- **형태**: 하이버 앱 내 "웹툰 주인공 코디" 큐레이션 탭 + 인스타그램 시리즈 콘텐츠
- **비용**: 거의 없음 (콘텐츠 제작만 필요, 물리적 상품 재고 불필요)
- **기대 효과**: 하이버 앱 체류 시간 증가 + 더그림 웹툰 유입
- **실행**: 더그림이 캐릭터 컷 제공 → 하이버 MD가 매칭 상품 선정 → 공동 SNS 발행

#### 🏷️ B. 한정판 콜라보 굿즈 입점

- **구조**: 더그림 캐릭터 디자인 적용 의류·액세서리 → 하이버 단독 판매
- **우선 타깃 상품**: 후드티, 반팔, 에코백 (웹툰 굿즈 최다 판매 카테고리)
- **더그림 현황**: 굿즈 매출 이미 52억 원 — 기존 상품 라인업에서 하이버 전용 선별 입점 가능
- **하이버 이점**: 플랫폼 화제성 회복 + 기존 없던 IP 상품 카테고리 신설
- **수익 배분**: 판매액의 30~40% 더그림 / 나머지 하이버 (입점 수수료 구조)

---

### Tier 2: 핵심 협업 (3~9개월)

#### 📱 C. 하이버 앱 내 웹툰 챕터 공개 (독점 선공개)

- **구조**: 더그림 신작 또는 스핀오프 챕터를 하이버 앱에서 1주일 선공개
- **하이버 이점**: 앱 방문 이유 생성 → MAU 증가 → 회생 과정에서 지표 개선
- **더그림 이점**: 새로운 독자 유입 경로 확보 (패션에 관심 있는 남성 타깃)
- **연계**: 해당 챕터 속 캐릭터 코디 → 하이버 상품 바로 연결 (커머스 연동)

#### 🎯 D. "외모지상주의 스타일" 기획전

- **구조**: 작중 에피소드나 캐릭터 변화(비포·애프터)에서 영감을 받은 시즌 기획전
- **예시**: "박형석 변신 룩", "배고은 데일리 스타일", "싸움독학 파이터 룩"
- **캠페인**: 웹툰 이미지 + 실제 상품 사진 나란히 배치하는 비주얼 광고
- **채널**: 하이버 앱 메인 배너 + 인스타그램 + 네이버웹툰 광고 지면 교차 집행

---

### Tier 3: 장기 전략 (9개월+)

#### 🤝 E. 박태준 유니버스 IP 기반 하이버 오리지널 웹툰

- **구조**: 하이버 세계관을 배경으로 한 스핀오프 단편 웹툰 (패션·스타일 테마)
- **내러티브**: "하이버에서 일하는 스타일리스트 주인공이 박태준 유니버스 캐릭터들과 엮이는 이야기"
- **배급**: 하이버 앱 독점 → 일정 기간 후 네이버웹툰 정식 연재
- **목적**: 하이버를 단순 쇼핑몰이 아닌 '콘텐츠 플랫폼'으로 포지셔닝

#### 🌐 F. 글로벌 확장 시 공동 진출

- 더그림이 글로벌 IP 유통 확장 시 하이버 글로벌몰 연계 (한류 패션 + 웹툰 세트)
- 외모지상주의 넷플릭스 시즌 2 등 대형 이벤트 타이밍에 글로벌 기획전

## 4. 딜 구조 권고안

### 단계별 접근

```
Phase 1 (탐색, 0~2개월)
└── 캐릭터 코디 콘텐츠 파일럿 (비용 거의 없음)
└── 성과 측정: 앱 유입, 소셜 반응

Phase 2 (검증, 2~6개월)
└── 굿즈 입점 계약
└── 하이버 앱 웹툰 선공개 파일럿
└── KPI: MAU 증가율, 굿즈 판매량, SNS 팔로워

Phase 3 (확장, 6개월+)
└── 오리지널 웹툰 기획
└── 정식 MOU → 장기 파트너십 계약
```

### 수익 배분 원칙

| 영역 | 더그림 | 하이버 |
|------|--------|--------|
| 굿즈 판매 | 40~50% (IP 로열티+제조) | 50~60% (유통·플랫폼) |
| 콘텐츠 제작 | 제작비 수취 | 제작비 지불 |
| 웹툰 선공개 | 고정 라이선스 피 | 앱 독점 효과 |

## 6. 왜 지금인가 — 타이밍 논리

**하이버 입장:**
- 기업회생 과정에서 브랜드 재활성화가 절실
- 마케팅 예산 없이 화제성을 만들 수 있는 IP 협업은 최적의 전략
- 선례 없는 웹툰 × 남성 패션 플랫폼 콜라보 → 언론·SNS 화제 확보

**더그림 입장:**
- 웹툰 독자 외 새로운 유통 채널 확보
- 굿즈 매출(52억) 확대 경로
- 하이버의 375만 남성 회원 = 새로운 독자 풀

**시장 맥락:**
- 웹툰 IP의 커머스 연동은 글로벌 표준 (Disney × Webtoon, Warner Bros × Webtoon)
- 국내에서 웹툰 × 패션 플랫폼 정식 콜라보 선례 없음 → **선점 효과**
