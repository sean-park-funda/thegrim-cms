## 역할별 요약

### admin (관리자)
전체 권한. 회계 포함 모든 기능 접근 가능.

### executive (임원)
- 콘텐츠/파일 조회만 가능
- 매출 메뉴 접근 가능
- 회계 데이터 및 보고서 조회 가능 (수정 불가)
- 사용자 관리 불가

### manager (매니저)
- 콘텐츠 생성/수정/삭제
- 파일 업로드/삭제 (전체)
- 공정 관리
- 사용자 초대/관리
- 매출 메뉴 접근 가능
- 회계 접근 불가

### staff (스태프)
- 콘텐츠 조회
- 파일 업로드 및 **본인 파일만** 삭제

### viewer (조회자)
- 콘텐츠/파일 조회 및 다운로드만 가능
- 수정/삭제/업로드 모두 불가

### accountant (회계)
- 매출 메뉴 접근 가능
- 회계 시스템 전담 (조회, 생성, 수정, 승인, 카테고리, 예산)
- 콘텐츠/파일 조회 가능
- 콘텐츠 생성/수정/삭제 불가

### strategy (전략팀)
- 회계를 제외한 모든 기능 접근 가능
- 콘텐츠 생성/수정/삭제, 파일 업로드/삭제, 공정 관리 포함
- **매출 메뉴 접근 가능** (매출 현황, 작품별 매출, 성장 분석, 랭킹, AI 채팅)
- 사용자 초대/관리는 불가 (admin 전용)
- **회계 메뉴(정산/거래/예산) 접근 불가**
