# 더그림 CMS — 제품 정의

## 제품 목적

웹툰 제작사 더그림의 내부 작업관리 시스템. 웹툰 제작 파이프라인(파일 관리, 컷/회차 구조, 캐릭터)을 디지털화하고, AI를 이용한 이미지 분석/재생성, 영상 생성을 지원한다.

## 타겟 사용자

더그림 내부 팀원 전용 (외부 공개 서비스 아님)

| 역할 | 주요 업무 |
|------|----------|
| Admin | 사용자 초대/관리, 전체 설정 |
| Manager | 작품/회차/컷 생성·수정·삭제, AI 기능 사용 |
| Staff | 파일 업로드/다운로드 |
| Viewer | 조회만 |
| Accountant | 회계·정산 시스템 전용 |

## 핵심 기능

### 1. 웹툰 파일 관리
- 웹툰 → 회차 → 컷 계층 구조
- 파일 업로드 (드래그앤드롭, 클립보드 붙여넣기)
- Gemini API 자동 이미지 분석 및 메타데이터 생성
- 메타데이터 기반 검색

### 2. AI 이미지 재생성
- 기존 이미지를 다양한 AI 스타일로 재생성
- 배치 재생성 지원
- 재생성 히스토리 관리

### 3. 영상 생성
- **Script-to-Movie**: 대본 → 컷(글콘티) → AI 이미지 → 영상 자동 생성
- **Script-to-Shorts**: 대본 → 쇼츠 영상
- **웹툰 애니메이션**: 기존 웹툰 컷에 애니메이션 적용
  - ComfyUI (LTX 2.3 22B) 로컬 5090 PC 연동
  - 세그먼트별 생성 후 FFmpeg 병합

### 4. 캐릭터 관리
- 캐릭터 시트 AI 자동 생성
- 3D 캐릭터 관계도 시각화 (Three.js)
- 시간축 관계 변화 추적

### 5. 회계·정산 시스템
- 거래 내역 관리 (수입/지출)
- 파트너·직원·계약 정산
- Excel 임포트 지원
- 작품별·월별 매출 분석

### 6. 기타
- 자유 창작 AI 챗 (Gemini)
- 몬스터 생성기
- 공지사항 관리
- 레퍼런스 파일 관리

## 비기능 요구사항

| 항목 | 요구사항 |
|------|----------|
| 인증 | 이메일+비밀번호, 초대 토큰 기반 회원가입 |
| 권한 | 역할별 접근 제어 (RLS + 클라이언트 권한 체크) |
| 영상 생성 | ComfyUI 큐 기반 처리 (5090 PC 단일 노드) |
| 배포 | Vercel (프론트) + AWS Lightsail (ComfyUI 릴레이 API) |
| 스토리지 | Supabase Storage (이미지, 영상) |
| DB | Supabase PostgreSQL |
| 언어 | 한국어 UI |
