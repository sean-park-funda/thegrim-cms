# 매뉴얼 스크린샷 가이드

이 폴더에는 각 기능의 스크린샷 이미지를 저장합니다.

## 이미지 파일명 규칙

각 기능별로 다음 파일명을 사용합니다:

- `webtoon-management-1.png`, `webtoon-management-2.png` - 웹툰 관리
- `process-management-1.png` - 공정별 관리
- `file-management-1.png`, `file-management-2.png` - 파일 관리
- `reference-files-1.png` - 레퍼런스 파일 관리
- `search-1.png` - 검색 기능
- `ai-image-analysis-1.png`, `ai-image-analysis-2.png` - AI 이미지 분석
- `ai-image-regeneration-1.png`, `ai-image-regeneration-2.png` - AI 이미지 재생성
- `monster-generator-1.png` - 몬스터 생성기
- `character-management-1.png`, `character-management-2.png` - 캐릭터 관리
- `character-pose-1.png`, `character-pose-2.png` - 캐릭터 자세 만들기
- `admin-features-1.png` - 관리자 기능

## 스크린샷 캡처 방법

1. 각 기능 페이지로 이동합니다 (예: `/webtoons`, `/processes`, `/admin` 등)
2. 브라우저 개발자 도구를 엽니다 (F12)
3. 원하는 화면을 보여주는 상태에서 스크린샷을 캡처합니다
4. 이미지를 `public/manual/` 폴더에 저장합니다

## 권장 스크린샷 내용

각 기능의 주요 화면을 캡처하세요:

- **웹툰 관리**: 웹툰 목록, 웹툰 생성 다이얼로그
- **공정별 관리**: 공정 목록 및 파일 그리드
- **파일 관리**: 파일 그리드, 파일 업로드 화면
- **레퍼런스 파일 관리**: 레퍼런스 파일 다이얼로그
- **검색 기능**: 검색 결과 화면
- **AI 이미지 분석**: 분석 결과가 표시된 파일 상세 화면
- **AI 이미지 재생성**: 재생성 작업 공간 화면
- **몬스터 생성기**: 몬스터 생성 화면
- **캐릭터 관리**: 캐릭터 목록, 캐릭터 시트 관리 화면
- **캐릭터 자세 만들기**: 3D 뷰어 화면, 변환 결과 화면
- **관리자 기능**: 관리자 페이지 화면

## 이미지 형식

- 형식: PNG 또는 JPG
- 권장 해상도: 최소 1920x1080 (Full HD)
- 파일 크기: 각 이미지당 2MB 이하 권장
